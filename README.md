# Foodie

Foodie is a simple web application for browsing recipes and generating shopping lists. Built with React, TypeScript, and Vite.

## Features
- Browse a selection of recipes
- View recipe details and ingredients
- Generate a shopping list from selected recipes

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure
- `src/` - React source code
- `recipes/` - JSON recipe data
- `public/` - Static assets
